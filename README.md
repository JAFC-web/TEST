# Instalación y ejecución

## 1. Instalar Node.js

Entrar a:

https://nodejs.org/

Descargar la versión:

```text
LTS
```

Durante la instalación:

* dejar todo por defecto
* activar la opción:

```text
Add to PATH
```

Finalizar instalación.

---

## 2. Verificar instalación

Abrir CMD y ejecutar:

```bash
node -v
```

y:

```bash
npm -v
```

Si aparecen versiones, Node.js quedó instalado correctamente.

---

## 3. Descargar el proyecto

Descargar el archivo ZIP del proyecto.

---

## 4. Descomprimir

Extraer el contenido del ZIP en cualquier carpeta.

---

## 5. Copiar la ruta

Abrir la carpeta del proyecto y copiar la ruta.

Ejemplo:

```text
C:\Users\user\Downloads\dealhunter-real-apis-fixed
```

---

## 6. Abrir CMD

Presionar:

```text
WIN + R
```

Escribir:

```text
cmd
```

y presionar Enter.

---

## 7. Entrar al proyecto

```bash
cd "C:\Users\user\Downloads\dealhunter-real-apis-fixed"
```

---

## 8. Instalar dependencias

```bash
npm install
```

---

## 9. Ejecutar el proyecto

```bash
npm run dev
```

---

## 10. Abrir en navegador

```text
http://localhost:5173
```
